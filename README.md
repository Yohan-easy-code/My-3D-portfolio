# Portfolio - Yohan Zouari

Portfolio personnel developpe avec React et Vite pour presenter mon positionnement de developpeur full-stack, mes concepts front-end, et mon approche entre branding, experience utilisateur, 3D et automatisation.

## Bio

Je suis Yohan Zouari, developpeur full-stack avec un focus sur React, Next.js, Node.js, TypeScript, Docker CI/CD, l'automatisation IA et l'orchestration d'agents.

Mon approche combine :

- une base front-end propre et modulaire
- des interfaces rapides, lisibles et orientees conversion
- une integration back-end claire pour les apps et les workflows metier
- l'automatisation IA quand elle apporte un vrai gain de productivite
- une sensibilite forte au design, a la 3D et a la presentation de marque

## A propos du projet

Ce repository contient mon site portfolio ainsi que plusieurs concepts de landing pages et de projets front-end. L'objectif n'est pas seulement de montrer du design, mais de rendre visible ma facon de construire une interface, structurer un produit et relier branding, code et execution technique.

### Stack du portfolio

- React 19
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Three.js
- React Three Fiber / Drei
- EmailJS

### 2. YouCode

Concept de plateforme de cours avec interface apprenant, gestion de contenu et vision produit plus structuree.

Technologies :

- React
- Next.js
- TypeScript
- Prisma
- TanStack Query
- Vitest

Competences acquises :

- structuration d'une interface applicative orientee produit
- conception de dashboards et de parcours admin
- decoupage de composants reutilisables et d'une UI plus modulaire
- reflexion autour des roles, du contenu et de la progression utilisateur
- utilisation des Server Components dans Next.js
- mise en place de l'authentification
- usage des Server Actions pour gerer des flux relies au serveur
- utilisation des Intercepting Routes pour affiner l'experience de navigation
- integration de Suspense pour mieux controler les etats de chargement
- gestion des donnees avec TanStack Query
- ecriture de tests avec Vitest
- manipulation de requetes Prisma pour structurer l'acces aux donnees
- utilisation de Server Components pour mieux separer rendu serveur et logique d'interface
- mise en place de systemes d'authentification
- usage des Server Actions pour gerer certaines mutations cote serveur
- travail avec les Intercepting Routes pour enrichir la navigation
- integration de Suspense pour mieux gerer le chargement et l'experience percue

Repository :

- `https://github.com/Yohan-easy-code/you-code-project`

## Lancer le projet

### Installation

```bash
npm install
```

### Developpement

```bash
npm run dev
```

### Build production

```bash
npm run build
```

### Verification lint

```bash
npm run lint
```
